# 能力库使用说明

## 能力库简介

能力库(Capability Library)存放了所有AI可能用到的skill和MCP工具，包括编程、文稿写作、文件整理、搜索研究等【这里可以放你所有能力库相关skill和MCP覆盖的领域】。它不在你的初始上下文中，但你可以从中获取完成任务所必须的相关能力。这些skill和MCP工具都是用户根据实际需求精心整理的，务必去主动了解。

能力库根目录位于"D:\path\to\capability-library"【这里应该替换为你的实际目录】

## 强制规则

每次新的任务之前，你都需要先使用以下命令从能力库中获取建议：
```powershell
cd "D:\path\to\capability-library"【这里应该替换为你的实际目录】
python tools/advisor/ability_suggest.py "<你的任务背景和需求>"
```

该命令会返回 JSON，里面包含推荐的 Skill 和 MCP。

- 如果返回 Skill，读取其中的 `path` 指向的 `SKILL.md`。
- 如果返回 MCP，先使用load命令加载 MCP，了解该MCP对应的所有工具和具体用途，再根据任务要求使用use命令调用具体工具。

如果你认为adviser提供的内容过少或不够用、不够清晰，可以从能力库的清单寻找可用的能力：

```text
skills/manifest.json
mcps/registry.json
```

## 使用 MCP

先加载 MCP：

```powershell
python tools/mcp/load_mcp.py --name "MCP名称或ID"
```

再调用工具：

```powershell
python tools/mcp/use_tool.py --mcp "MCP名称或ID" --tool "工具名" --params-json "{...}"
```

参数复杂时，使用 JSON 文件：

```powershell
python tools/mcp/use_tool.py --mcp "MCP名称或ID" --tool "工具名" --params-file params.json
```

`load_mcp.py` 默认会保持会话，后续 `use_tool.py` 可以复用连接。

所有命令都应在 `<能力库根目录>` 下运行。
